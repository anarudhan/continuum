# Using Continuum Guidelines with Cursor


> **Brand:** This document follows the [Continuum Circuit Board Brand System](branding/BRAND_SYSTEM.md).
> Colors: Deep Navy `#0A1628`, Electric Cyan `#00D4FF`, Soft Lavender `#A78BFA`.


This repository includes a committed Cursor project rule (`.cursor/rules/continuum-guidelines.mdc`) so the same guidelines apply when you open the project in Cursor.

## Setup

The rule is already committed at `.cursor/rules/continuum-guidelines.mdc`. Cursor will automatically load it when you open this project.

## Using in Other Projects

To use these guidelines in other projects:

1. Copy `.cursor/rules/continuum-guidelines.mdc` to your project's `.cursor/rules/` directory
2. Or copy the content into Cursor's global rules (Settings > General > Rules for AI)

## Relationship to CLAUDE.md

- `CLAUDE.md` is the canonical source for Claude Code
- `.cursor/rules/continuum-guidelines.mdc` is the Cursor-native format
- Both contain identical guidelines, just formatted for their respective tools
- When updating guidelines, update BOTH files to keep them in sync
